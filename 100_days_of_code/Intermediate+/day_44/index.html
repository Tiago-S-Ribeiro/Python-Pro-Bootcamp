Technical Skills

HAProxy

HAProxy is a very fast and reliable reverse-proxy offering high availability, load balancing, and proxying for TCP and HTTP-based applications. It's ideal for very high traffic web sites and powers a significant portion of the world's most visited ones.

Saltstack

Salt is a configuration management and orchestration tool. It has the capability of automating repeated administrative and code deployment tasks. It's multithreaded design enables the execution of thousands of simultaneous tasks.

Redis

Redis is an open source, in-memory data structure store, used as a database, cache, and message broker. It provides data structures such as strings, hashes, lists, sets, sorted sets with range queries, bitmaps, hyperloglogs and geospatial indexes.
